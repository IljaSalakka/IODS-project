Diagnostic plots

Because I created multiple models, I will produce the diagnostic plots for the starting point (model 1) and the final model (model 3). The regression model assumptions inspected here are constant variance and linearity, normality (of residuals) and lack of too influential outliers.

plot(model1, which = c(1,2,5))

plot(model3, which = c(1,2,5))

As can be seen from the plots, these two models do not differ much from each other in model diagnostics. Residual vs fitted plot shows that aside 3 (kind of) outliers, the variance is quite constant across fitted values. There is some decrease in variance when fitted values are high, but this is based on just few observations so I dont see it as a big problem. Q-Q plot do not show clear deviance from normality. It is normal to see some deviance in end points and this does not pose any real problem for model assumptions. The residuals vs leverage plot does show us that there are no single outliers that are too influential to the model. Overall, based on visual inspection of diagnostic plots, assumptions for regression model were not violated.
